# Snake-and-Ladder
![image](https://user-images.githubusercontent.com/75296055/195662210-1bb3a4b2-4f5c-4b8a-a651-05d34de98290.png)

Snakes and Ladders is a chance-based board game featuring 100 squares. Players have to get to the top while dealing with the consequences of every dice roll. Historically, the game was used to teach children basic moral values. The ladders were representative of virtues while the snakes of vice. Today, you could see it as a metaphor for lifeâ€™s journey with all of its ups and downs. This is Python-based Snake and ladders game.
